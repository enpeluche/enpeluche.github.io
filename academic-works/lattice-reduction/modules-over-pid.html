Modules sur un anneau principal

Dans la suite, on considère un anneau principal $A$, c’est-à-dire un anneau (commutatif unitaire) dans lequel tout idéal est principal.
Soit $M$ un module libre sur un anneau principal $A$. Alors tout sous-module de $M$ est également libre et son rang est inférieur ou égal à celui de $M$.
Dans le $\Z$-module libre $\Z^2$, considérons le sous-ensemble \[ N \;=\; \{(a,b)\in \Z^2 \mid a \equiv b \pmod{10}\}. \] On constate que $N$ est un sous-module de $\Z^2$, et qu’il est engendré par les vecteurs $(1,1)$ et $(0,10)$. Ceux-ci sont $A$-linéairement indépendants, donc $N$ est libre de rang $2$, tout comme $\Z^2$ lui-même.
Commentaires sur les exemples :
  1. \(\K[x]^n\) est le module libre canonique : c’est un \(\K[x]\)-module libre de rang \(n\). Il sert de modèle aux réseaux polynomiaux.
  2. \(\K[x]/(x^n)\) est un module de torsion : tout élément est annulé par une puissance de \(x\). Il n’admet pas de base libre.
  3. \(\K[x]^\infty\) (somme directe infinie) est un module libre, mais non de type fini. Il possède une base infinie indexée par \(\N\).
  4. \(\K(x)\), le corps des fractions rationnelles, est un module divisible mais non libre. Il contient des éléments sans expression unique comme combinaison de base.
  5. $(\K[x])^n / (x \cdot \K[x])^n$ est un module quotient, utilisé dans les algorithmes de bases d’ordre modulo $x^\sigma$. C’est un module de torsion.
